The text you type in will get sent to the server application using WebSockets.
The server application will use a WebSocket decoder that the application defined. This decoder works by changing the message to all upper case.
The server application will then send the decoded message back using a WebSocket encoder that the application defined. This encoder will map vowels to numbers.
Type 'stop' to tell the server application to close down the session. The stop message will not be decoded nor the return message encoded.
